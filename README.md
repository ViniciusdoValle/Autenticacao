# Autenticacao
Este projeto é um conjunto de telas de autenticação de usuário (login, cadastro e recuperação de senha), desenvolvido com HTML, CSS puro e um pouco de JavaScript embutido no HTML como forma de prática e estudo para aprimorar minhas habilidades em desenvolvimento front-end.


Login:

• Campo para inserir: nome de usuário, e-mail ou número de telefone associado à conta.

• Campo de senha com ícone "olhinho" para mostrar/ocultar o conteúdo.

• Ao preencher corretamente e clicar em "Entrar", aparece uma mensagem: "Você entrou".

• Inclui uma caixa de seleção para que o usuário possa optar por manter-se conectado após o login.

• Link para se cadastrar e link para redefinir senha.


Recuperar a Senha: 

• Campo para digitar o e-mail usado no cadastro.

• Ao clicar em "Enviar", exibe: "Enviamos um link de redefinição para o e-mail informado."

• Link para voltar à tela de login.


Cadastro: 

• Campos: nome completo, data de nascimento, nome de usuário, e-mail, telefone e senha.

• Validação de idade: somente maiores de 18 anos conseguem criar uma conta.

• Se o usuário for menor de idade, aparece um alerta: "Você precisa ter 18 anos ou mais para criar uma conta."

• Também conta com o ícone de olho para exibir/ocultar a senha.

• Link para voltar à tela de login.
