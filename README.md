# Autenticacao
Projeto com três páginas principais: Tela de Login, Tela de Cadastro e Tela de Recuperação de Senha, desenvolvidas com HTML5, CSS3 e um pouco de JavaScript inserido no HTML para interações como exibição de senha e validação de formulário.

1 - Tela de Login:

- Permite que o usuário entre com nome de usuário, e-mail ou telefone, além da senha.

- Possui a opção "Manter-me conectado".

- Ao clicar em "Entrar", exibe um alerta informando que o login foi realizado com sucesso (caso os dados sejam válidos).

- Contém dois links: "Não tem uma conta? Cadastre-se": Leva o usuário para a tela de cadastro. "Esqueceu a sua senha?": Redireciona para a tela de recuperação de senha.

2 - Tela de Cadastro:

- Formulário com os seguintes campos: Nome completo, idade (somente maiores de 18 anos podem se cadastrar), nome de usuário (com validação de caracteres), e-mail, telefone e senha.

- Ao tentar cadastrar com idade inferior a 18 anos, é exibido um alerta avisando que para criar uma conta o usuário precisa ser maior de idade e a conta não é criada.

- Cadastro válido exibe um alerta de sucesso e limpa o formulário.

3 - Tela de Redefinição de Senha:

- Permite ao usuário solicitar a redefinição de senha informando apenas o seu e-mail.

- Após clicar em "Enviar", exibe um alerta informando que um link de redefinição foi enviado para o e-mail fornecido.

- Link de retorno para o login: "Voltar para o login".
